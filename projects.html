Featured Projects

Python Agricultural Calculator

Developed a comprehensive Python-based calculator application that helps farmers with:

  • Crop yield estimations based on land area and historical data
  • Fertilizer requirement calculations for different crops and soil types
  • Profit margin analysis considering input costs and market prices
  • Water requirement estimations for irrigation planning
  • Feed conversion calculations for livestock management

This tool has been used by over 50 local farmers to optimize their resource allocation and improve productivity.

Technologies: Python, SQLite for data storage, Command-line interface

Farm E-Commerce Platform

Created an online marketplace that connects farmers directly with consumers, eliminating middlemen and improving farmer profits while providing consumers with fresh, locally-sourced agricultural products.

Key features include:

  • Farmer profiles and product listings with detailed information
  • Secure payment processing and escrow system
  • Delivery scheduling and logistics coordination
  • Rating and review system to build trust
  • Analytics dashboard for farmers to track sales and customer preferences

Technologies: HTML, CSS, JavaScript, PHP, MySQL

Farm Management System

A comprehensive web application designed to help farmers manage their operations efficiently, track resources, and monitor productivity remotely.

The system includes modules for:

  • Crop planning and rotation scheduling
  • Livestock tracking and health monitoring
  • Inventory management for seeds, fertilizers, and equipment
  • Financial record-keeping and expense tracking
  • Weather integration for informed decision-making
  • Task management and labor allocation

This system has been implemented on 5 medium-sized farms in Kenya with positive feedback on improved organization and productivity.

Technologies: HTML, CSS, JavaScript, Python (Django), PostgreSQL

Agricultural Data Collection Tool

Features include:

  • Photo and video documentation capabilities

  • Basic statistical analysis and visualization tools
  • Export functionality in multiple formats (CSV, Excel, JSON)

Technologies: ODK, KoBoToolbox, Python for data processing

Research Projects

Climate-Resilient Farming Practices Study

Participated in a research project with KCEP-CRAL to document and analyze climate-resilient farming practices across different agro-ecological zones in Kenya. My role included data collection, farmer interviews, and helping analyze the effectiveness of different adaptation strategies.

Key outcomes:

  • Compiled database of over 30 climate adaptation techniques
  • Developed assessment framework for technique effectiveness
  • Co-authored field report used by agricultural extension officers

Mobile Technology Adoption in Rural Farming

Conducted a survey study on factors affecting mobile technology adoption among smallholder farmers. The research identified key barriers and enablers for digital agriculture tools in rural communities.

Key findings:

  • Literacy levels and interface design significantly impact adoption rates
  • Peer demonstration is more effective than formal training
  • Value proposition must be demonstrated within one growing season
